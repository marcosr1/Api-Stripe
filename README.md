# 🚀 API de Pagamentos com Stripe

API REST desenvolvida em Node.js para gerenciamento de pagamentos utilizando o Stripe.

## 📦 Tecnologias

- Node.js
- Express
- Stripe
- PostgreSQL
- Sequelize
- Socket.IO
- JWT

## ⚙️ Funcionalidades

- Criar pagamentos
- Processar checkout
- Webhooks do Stripe
- Gerenciamento de clientes
- Sistema de pedidos
- Atualização de status em tempo real

## 💳 Integração de pagamentos

A API utiliza a plataforma **Stripe** para processar pagamentos.

Suporte para:

- Cartão de crédito
- Carteiras digitais
- Métodos locais (quando disponíveis)
